Design Document Suite

The Design Document Suite or 'DDS' is a document automation and quality assurance application for the Procter & Gamble account at the WestRock Merchandising Displays division. The Application focuses on uploading Excel documents exported in HTML format with developed UI for users to link project based customer data to appropriate ‘cells’, eliminating redundant copy+paste keystrokes and improving quality. (JavaScript/AJAX, PHP, MSSQL.).

• Built from the ground up, responsible for implementation and innovation of customer documentation process.
• Responsible for support of application via ticketing system (ServiceNow).
• Responsible for database management from internal & customer resources.
• Responsible for project metric reporting from generated data within application.


* Prototype demo available. Source code kept private by request of the company.

Ghost Link was a group senior capstone project for the Cybersecurity Program at the University of Cincinnati.
The project won the Western & Southern Award for IT Expo Presentation in Cybersecurity.

This tool is an all-in-one URL scanning service that detects malicious websites using powerful file and network scanners.
Maliciousness scores are appended to the outputs of the site's analyzers, quickly helping analysts determine if a URL is malicious or not.

Problem & Solution:
• Security professionals are required to use several tools daily to analyze URLs.
• Using multiple tools for analysis decreases efficiency.
• Most URL scanning tools do not utilize file scanning.
• Ghost Link centralizes URL and File analysis, allowing security professionals to become more efficient.


* This demo is front-end only and is not connected to a server or api.
